# meetup-friends
Meetup-friends is an userscript to manage a friend list on (meetup.com)

## Requirements
* Firefox user : [Greasemonkey](https://addons.mozilla.org/fr/firefox/addon/greasemonkey/)
* Chrome user : it should works as it :)
